# Project Hermes GitHub Organization

As the heading suggests, this is the GitHub Organization for [Project Hermes][0] and is where any and all code related to the project will be stored.

## Project Repositories

This section provides information on all of the repositories that make up the project and what purpose they serve.

### API

- [GitHub Repository][1]
- [Documentation][2]

### Mobile App

- [GitHub Repository][3]
- [Documentation][4]

### Docs

> This repository is home to the [documentation website][10], created with [Material for MkDocs][9], for all of **Project Hermes**.

- [GitHub Repository][11]
- [Website][10]

### Website

> This repository is home to the website portal for interacting with Project Hermes.

- [GitHub Repository][5]
- [Documentation][6]

### OpenAPI

> This repository is home to the OpenAPI Specification files for the Hermes API and serves as a Stoplight Workspace for a GUI method of managing the OpenAPI Specification.

- [GitHub Repository][12]
- [Swagger UI][13]

### Shared Lib

> This repository is home to an NPM package that provides a number of helpful interfaces, utility classes, etc., that are used throughout Project Hermes.

- [GitHub Repository][7]

## Websites/URLs

The following is a list/table of all the websites/URLs that are publicly available:

| Name             | URL                           | Description                                                                                                                           |
| ---------------- | ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| API Swagger Docs | https://swagger.hermesapp.dev | A website, built with [Swagger UI][8], that provides documentation on the API.                                                        |
| Project Docs     | https://docs.hermesapp.dev    | A website, built with [Material for MkDocs][9], that provides documentation for the entire project.                                   |
| Project Homepage | https://projecthermes.dev     | A website, built with Material for MkDocs, that serves as the landing page for the project. Not sure this will stick around though... |
